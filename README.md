# AutoChessRogue

自走棋 × 肉鸽爬塔策略游戏，基于 Godot 4 开发。

## 玩法

**布阵 → 选关 → 自动战斗 → 结算**，循环5波，挑战终局 Boss。

- **备战席**：购买、升级、摆放棋子，最多5个单位上场
- **关卡选择**：每波从普通/困难两个选项中选一个，第5波固定 Boss 战
- **自动战斗**：无需操作，棋子按攻速自动移动、攻击
- **结算**：胜利获得金币，失败扣生命值，生命值归零则游戏结束

## 单位

| 名称 | 定位 | 费用 | 技能 |
|------|------|------|------|
| 骑士 | 近战坦克 | 5 | — |
| 战士 | 近战输出 | 5 | 撼地（AOE + 护甲击碎） |
| 游侠 | 远程物理 | 5 | 蓄力一击（穿透直线伤害） |
| 法师 | 远程魔法 | 5 | — |
| 牧师 | 远程治疗 | 20 | 快速治疗（单体治疗 + 防御 buff） |

## 战斗规则

- 物理伤害 = `AD × 100 / (100 + ARM)`
- 魔法伤害 = `技能值 × 100 / (100 + MDF)`
- 暴击：25% 概率，伤害 × 125%
- 近战单位向最近敌人直线移动，到达攻击距离后停止
- 远程单位原地攻击范围内最近目标

## 技术栈

- **引擎**：Godot 4.6
- **语言**：GDScript
- **架构**：BattleEngine 战斗逻辑与渲染分离，通过信号驱动动画层

## 项目结构

```
autoload/       # 全局单例（GameState / BattleEngine / DataManager）
core/           # 核心逻辑（Unit / DamageCalc / SkillSystem）
scenes/         # 游戏场景（备战席 / 关卡选择 / 战斗 / 结算 / 结束）
ui/             # UI 组件（棋盘 / 商店卡 / 属性面板 / 血条）
data/           # 配置数据（单位 / 敌人 / 波次 / 技能）
assets/         # 游戏资源
```
